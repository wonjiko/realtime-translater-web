## 1. 중간 번역 검증

### 1-1. 기본 동작
- [ ] 음성 언어를 특정 언어(예: English)로 설정
- [ ] REC 시작 후 긴 문장 말하기
- [ ] 말하는 도중 ~1초 후 중간 번역 엔트리가 표시되는지 확인
- [ ] 중간 엔트리가 반투명 + 이탤릭 + 왼쪽 테두리로 시각 구분되는지 확인
- [ ] 중간 엔트리에 "Translating..." → 번역 결과로 업데이트되는지 확인

### 1-2. Final 전환
- [ ] 발화가 끝나면 (브라우저가 isFinal 판정) 중간 엔트리가 사라지는지 확인
- [ ] 최종 엔트리가 정상 스타일(불투명, 직선체)로 표시되는지 확인
- [ ] 최종 번역이 별도로 정상 수행되는지 확인

### 1-3. 빠른 연속 발화
- [ ] 짧은 문장을 연속으로 빠르게 말하기
- [ ] 중간 엔트리가 중복 생성되지 않는지 확인
- [ ] 각 final 결과마다 정상 엔트리가 생성되는지 확인

### 1-4. 녹음 중지 시 정리
- [ ] 말하는 도중 STOP 버튼 클릭
- [ ] 중간 엔트리와 interim 텍스트가 모두 제거되는지 확인
- [ ] 콘솔에 에러 없는지 확인

### 1-5. Whisper 모드 영향 없음
- [ ] Voice를 "Auto-Detect"로 설정 (OpenAI API Key 필요)
- [ ] 녹음 시작 → 발화
- [ ] 기존 Whisper 동작이 정상 작동하는지 확인 (중간 번역은 미적용)

## 2. 버튼 통합 + 녹화 상태 검증

### 2-1. 토글 동작
- [ ] 초기 상태: 버튼에 "REC" 표시, 빨간 배경
- [ ] 클릭 시 녹음 시작, 버튼 텍스트가 "STOP"으로 변경
- [ ] 다시 클릭 시 녹음 중지, 버튼 텍스트가 "REC"으로 복귀
- [ ] Stop 버튼이 별도로 존재하지 않는지 확인

### 2-2. 녹화 상태 시각 피드백
- [ ] 녹음 중 빨간 glow(box-shadow) 펄스 애니메이션이 보이는지 확인
- [ ] 기존 대비 녹화 상태가 확실히 눈에 띄는지 확인
- [ ] 녹음 중지 시 glow 효과가 즉시 사라지는지 확인

### 2-3. ReadOnly 모드
- [ ] 공유 URL로 접속 (ReadOnly 모드)
- [ ] 컨트롤 바가 숨겨져 있는지 확인
- [ ] 콘솔에 btnStop 관련 에러 없는지 확인

### 2-4. 언어 변경 중 재시작
- [ ] 녹음 중 Voice 언어 변경
- [ ] 녹음이 자동 재시작되고 버튼이 "STOP" 상태 유지하는지 확인
